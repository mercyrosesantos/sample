HTML Images

<img src="https://place-puppy.com/300x300">

An image can be shown using <img> tag along with its src attribute.

<img src="./images/image1.jpg" height="250" width="250">

Valid image extensions are the following:

  1. png
  2. jpg/jpeg
  3. bmp
  4. svg
  5. gif
  6. tiff
  7. webp
